# Shehr Scenes

Shehr Scenes is a web platform connecting people across Pakistan to cultural, social, and educational events. It focuses on improving accessibility for students and underinformed communities, while giving entrepreneurs and organizers a platform to showcase their events to a wider audience.

Live Demo:

## Key Features

- City-based browsing for discovering events in different regions
- Event categories and filters for easier navigation
- Community pages that highlight local engagement and groups
- Signup and login flow for user access and personalization

## Tech Stack

- HTML
- CSS

## How to Run Locally

1. Clone or download this repository to your computer.
2. Open `index.html` in your preferred web browser.
3. The site will redirect to `intro.html`, the welcome page.
4. Navigate through the website using the navigation menu.
5. Explore cities, events, and communities across Pakistan.

## Navigation Overview

- **Home**: Main landing experience with featured events and calls-to-action
- **Cities**: Browse events by major Pakistani cities (Lahore, Karachi, Islamabad, Peshawar, Sialkot, Multan)
- **Events**: Full event listing with category filters (music, education, fashion, sports)
- **Communities**: Discover community groups (women entrepreneurs, startups, cultural organizations)
- **Profile**: User profile management and contact information

## Features by Page

- **intro.html**: Welcome screen with Urdu title and entry button
- **explore.html**: Three-column layout with cities, events preview, and login/signup
- **home.html**: Hero section, feature highlights, and upcoming events
- **events.html**: Full event grid with filter options
- **communities.html**: Community cards showcasing different groups
- **City pages**: Individual city pages with location-specific events
- **profile.html**: User profile, contact form, and customer support information

## Browser Compatibility

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
